# Release Criteria — Base Madre Ultimate 3.0.0

La release 3.0.0 può essere chiusa solo se rispetta tutti questi criteri.

## Criteri tecnici

- build completata senza errori;
- lint senza errori;
- TypeScript senza errori;
- controlli performance superati;
- controlli sicurezza superati;
- controlli accessibilità superati;
- controlli release superati;
- working tree pulito;
- tag finale creato e pushato.

## Criteri architetturali

- struttura cartelle chiara;
- separazione tra core, UI, contenuti, configurazioni e moduli business;
- assenza di duplicazioni inutili;
- componenti riutilizzabili;
- moduli business non accoppiati tra loro.

## Criteri professionali

- foundation adattabile a più progetti;
- documentazione sufficiente per riprenderla in futuro;
- nessuna dipendenza inutile;
- nessun codice decorativo non necessario;
- nessuna soluzione fragile;
- nessun file senza scopo.

## Criterio finale

La Base Madre Ultimate 3.0.0 deve essere pronta per diventare il punto di partenza stabile dei futuri siti reali.
