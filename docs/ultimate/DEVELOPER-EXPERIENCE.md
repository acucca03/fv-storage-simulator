# Developer Experience — Base Madre Ultimate

La Base Madre Ultimate deve essere facile da riprendere, controllare, adattare e consegnare.

## Obiettivi

- comandi chiari;
- documentazione operativa;
- verifiche automatiche;
- onboarding rapido;
- struttura leggibile;
- progressi tracciati;
- rilascio controllato;
- base riutilizzabile per progetti reali.

## Comandi principali

Sviluppo locale:

    pnpm dev

Build:

    pnpm build

Check base:

    pnpm check

Healthcheck completo:

    pnpm health

Check completo release:

    pnpm check:full

## Verifiche automatiche

La foundation include verifiche per:

- struttura;
- architettura;
- configurazione;
- design system;
- moduli business;
- SEO;
- performance;
- stabilità;
- sicurezza;
- affidabilità;
- accessibilità;
- release;
- developer experience.

## Regola

Nessun blocco importante è completo finché:

- i controlli passano;
- il commit è stato creato;
- il push è stato completato;
- il working tree è pulito.
