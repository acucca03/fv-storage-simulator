# Changelog

## v0.1.0 - Initial Stable Foundation

Prima versione stabile di Site Foundation.

### Incluso

- Progetto Next.js pulito
- TypeScript
- Tailwind CSS
- pnpm
- Repository privato
- Configurazione centrale
- Metadata centralizzati
- Robots disallow globale
- Security headers base
- .env.example
- Layout base
- Header
- Footer
- MainLayout
- Componenti UI fondamentali
- Componenti form base
- SkipLink
- Pagina 404
- Pagina errore
- Utility cn
- Struttura contenuti generici
- Documentazione tecnica completa
- Checklist operative
- Procedura duplicazione base
- Script pnpm check
- Script pnpm check:full
- Verifica automatica struttura fondazione

### Obiettivo

Questa versione rappresenta la prima base madre privata, solida e riutilizzabile per futuri siti professionali.

### Nota

Site Foundation non deve essere deployato pubblicamente.

I siti reali devono nascere come progetti separati derivati da questa base.
